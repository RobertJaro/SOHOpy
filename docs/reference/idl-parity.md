# IDL Parity

The LASCO port should be validated against SolarSoft/IDL outputs whenever
possible.

## Current Script

```bash
scripts/idl/run_lasco_parity.sh tests/fixtures/idl/lasco_parity.json
```

This assumes an IDL environment where SolarSoft/LASCO routines are already on
the path.

## Fixture Policy

- Small synthetic fixtures may be committed.
- Large FITS files should be downloaded or generated by documented scripts.
- IDL output fixtures should include the IDL command, source routine versions,
  and enough metadata to reproduce the run.

## Priority Parity Targets

- calibration factors for representative C2/C3 filters and polarizers;
- missing-block string maps and absolute block numbers;
- read-port rectification orientation;
- Level 0.5 and Level 1 statistics;
- `DO_POLARIZ` tB/pB with `/PTF`;
- `c2_warp` and `c3_warp`;
- `adjust_hdr_tcr` time/roll/sun-center correction.
